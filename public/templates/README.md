# Salon Templates

This folder contains downloadable PDF templates for the Free Templates page.

## Current Files

1. **client-intake-form.pdf** - Client information and consultation form
2. **sanitization-checklist.pdf** - Daily cleaning and sanitation checklist
3. **service-menu-template.pdf** - Service menu pricing template

## Replacing Placeholder Templates

The current files are simple placeholders. To create professional templates:

### Option 1: Design in Canva/Adobe
1. Design professional PDF templates in Canva, Adobe InDesign, or similar
2. Export as PDF
3. Replace the files in this folder with the same filenames

### Option 2: Use Template Services
- Hire a designer on Fiverr/Upwork to create professional templates
- Use template marketplaces (Creative Market, Etsy) for pre-made salon forms
- Ensure files are printable, editable, and professionally branded

### Template Specifications

**Client Intake Form** should include:
- Contact information fields
- Emergency contact
- Medical history / allergies
- Previous treatments / hair history
- Desired results / consultation notes
- Terms of service agreement
- Signature lines

**Sanitization Checklist** should include:
- Opening procedures
- Workstation cleaning checklist
- Tool sterilization log
- Common area maintenance
- Closing procedures
- Staff sign-off sections

**Service Menu Template** should include:
- Service categories (Cut, Color, Style, Treatments)
- Service descriptions
- Duration estimates
- Pricing (or blank fields for pricing)
- Add-on services section
- Professional layout with space for salon logo

## Technical Notes

- Files must be named exactly as shown above
- Recommended file size: Under 5MB for fast downloads
- Format: PDF (standard/A4 or US Letter size)
- Ensure proper print margins (0.5" minimum)
